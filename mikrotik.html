Case Study

Hotspot Management System

Detail pengerjaan sistem manajemen jaringan hotspot menggunakan MikroTik untuk optimasi akses internet publik yang aman dan stabil.

Ringkasan Project

Project ini berawal dari kebutuhan sebuah kafe yang ingin membatasi durasi pemakaian internet pelanggan tanpa harus mengganti password Wi-Fi secara manual setiap hari. Saya mengimplementasikan sistem RADIUS dan User Manager pada MikroTik.

Apa yang Saya Kerjakan?

  • Konfigurasi Firewall dan NAT untuk keamanan jaringan internal.
  • Pembuatan Landing Page Hotspot kustom yang responsif.
  • Limitasi Bandwidth (Queue) per user agar koneksi adil.

Teknologi

MikroTik OS Winbox HTML/CSS
